# Лабораторная работа №8: Адаптивная вёрстка: Grid

## Описание

В данной лабораторной работе студентам предлагается ознакомиться с возможностями CSS Grid — современного инструмента для создания гибких и адаптивных макетов веб-страниц.

## Цели работы

- Освоить основы использования CSS Grid для построения сеточных макетов.
- Научиться создавать адаптивные макеты, которые корректно отображаются на различных устройствах.
- Ознакомиться с практическими примерами реализации макетов: сетка 3x3, структура страницы и адаптивная галерея.

## Задачи

1. Создать HTML-страницу, включающую описание и демонстрацию следующих макетов:
   - Простая сетка 3x3.
   - Макет страницы с боковой панелью, основным контентом и шапкой/подвалом.
   - Адаптивная галерея изображений, которая изменяет количество колонок в зависимости от ширины экрана.

2. Создать и подключить CSS-стили, использующие CSS Grid для оформления каждого макета.

## Структура проекта

- `index.html` — основной файл с HTML-разметкой, содержащей примеры макетов.
- `grid.css` — стили для первых макетов.
- `gridPractice.css` — стили для практических примеров, включая адаптивную галерею и макет страницы.
- `gridPractice.html` — практические задачи

## Основные компоненты

### 1. Простая сетка 3x3
- Использует `display: grid` с `grid-template-columns: repeat(3, 1fr)`.
- Обеспечивает равномерное распределение элементов по сетке.

### 2. Макет страницы
- Включает шапку, боковую панель, основной контент и подвал.
- Использует `grid-template-areas` для определения расположения элементов.
- Обеспечивает стабильную структуру при различных размерах окна.

### 3. Адаптивная галерея
- Использует `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))`.
- Адаптируется под ширину экрана, автоматически меняя количество колонок.
- Визуально выделена с помощью эффектов при наведении.

## Общая идея

| **Критерий** | **Flexbox** | **CSS Grid** |
|--------------|--------------|--------------|
| Тип компоновки | Одномерная | Двумерная |
| Управление | Строка или столбец | Строки и столбцы |
| Основное назначение | Выравнивание элементов | Построение макетов |
| Подходит для | Компоновки | Страниц и сеток |

**Вывод:**  
Flexbox лучше подходит для **линейных структур**, Grid — для **табличных и сложных**.
